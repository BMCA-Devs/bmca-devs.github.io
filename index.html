AI Coders 

You may ask our (well, maybe this opinion is mine) opinion on Coding AIs: Well, I think the future just cannot exist without AI; we are already in the AI time, and there is no way out now. So I think we should just embrace things as it is, and I actually do get help from various AI. I think the most important quote is that "AI is a tool to help you, not to replace you".
We must keep this in mind in the present and future, instead of not using AI completely or overusing it.
What is the best Coding AI, in my opinion? GitHub Copilot, for sure. I once asked both GitHub Copilot and ChatGPT and CodeGPT the same challenge, and GitHub copilot was fast, 
accurate, and explained how the code worked while other AIs had logic flaws, and no explaination.
